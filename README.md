# Sales Performance Dashboard (Power BI)
This project analyzes sales data to identify key business insights such as revenue trends, category performance, and regional distribution.

## Key Features
- KPI cards for Total Sales, Total Profit, and Total Orders
- Sales trend analysis over time
- Category-wise and region-wise performance
- Profit distribution visualization
- Interactive filtering using slicers

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization
- DAX

## Outcome
Generated insights to support data-driven decision-making.
