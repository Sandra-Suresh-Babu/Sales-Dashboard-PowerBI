# 📊 Sales Performance Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive sales dashboard built using Power BI to analyze business performance and generate actionable insights.

The dashboard focuses on key metrics such as revenue, profit, and order trends to support data-driven decision-making.

---

## 🚀 Key Features

* KPI cards for Total Sales, Total Profit, and Total Orders
* Sales trend analysis over time (time-series visualization)
* Category-wise and region-wise performance comparison
* Profit distribution using pie chart
* Interactive filtering using slicers

---

## 🛠 Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Basic Measures & Calculations)

---

## 📈 Insights Generated

* Identified top-performing product categories
* Analyzed sales trends across time periods
* Compared regional sales performance
* Evaluated profit contribution by category

---

## 🎯 Business Impact

This dashboard helps stakeholders:

* Monitor sales performance in real-time
* Identify growth opportunities
* Make data-driven business decisions
